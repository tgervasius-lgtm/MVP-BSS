BSS Smart Systems više nije statički Demo 3.0 bez backenda.

Aktualne upute za lokalno pokretanje, testiranje i produkcijsku pripremu nalaze se u README.md.
Backend runbook nalazi se u backend/OPERATIONS.md, a strojni API ugovor u openapi/bss-mvp-api-v1.yaml.
Aktualni dubinski audit nalazi se u BSS_PRODUCTION_READINESS_AUDIT.md.

Cloudflare Pages prikazuje samo statički shell. Funkcionalna aplikacija zahtijeva Node.js/Fastify servis i PostgreSQL na istom sigurnom originu.
